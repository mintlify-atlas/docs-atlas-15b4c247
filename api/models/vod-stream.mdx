## Overview

The `VodStream` model represents video-on-demand (VOD) content in the application. It uses Laravel Scout for search functionality and integrates with Spatie Laravel Data.

**Model Path:** `app/Models/VodStream.php`  
**Table:** `vod_streams`  
**Primary Key:** `stream_id` (non-incrementing)

## Type Casts

The model casts the following attributes:

- `stream_id` → `integer`
- `rating_5based` → `decimal:1`
- `is_adult` → `boolean`
- `added` → `immutable_datetime`

## Relationships

### Watchlists

Polymorphic one-to-many relationship with the Watchlist model.

```php
public function watchlists(): MorphMany
```

Returns all watchlist entries for this VOD stream.

**Usage:**
```php
$vod = VodStream::find(123);
$watchlists = $vod->watchlists;
```

## Search

The VodStream model uses Laravel Scout for full-text search. Only the `name` field is indexed.

**Example:**
```php
$results = VodStream::search('Inception')->get();
```

## Data Transfer Object

The model integrates with `VodStreamData` DTO class:

```php
protected $dataClass = VodStreamData::class;
```

## Adult Content Filtering

Use the `is_adult` flag to filter mature content:

```php
// Get non-adult content only
$familyFriendly = VodStream::where('is_adult', false)->get();

// Get adult content
$matureContent = VodStream::where('is_adult', true)->get();
```
