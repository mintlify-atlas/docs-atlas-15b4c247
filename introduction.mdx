## What is Lionz IPTV Downloader?

Lionz IPTV Downloader is a modern, high-performance IPTV content manager that integrates seamlessly with [Lionz TV](https://lionz.tv) IPTV service. Built with Laravel and React, it leverages the Xtream Codes API to fetch and manage Series and VODs, uses aria2c for efficient downloading, and MeiliSearch for lightning-fast content search and indexing.

## How It Works

1. **Connect** to Lionz TV using your Xtream Codes API credentials
2. **Sync** media content from the Lionz TV catalog to your local database
3. **Search** through your content library with lightning-fast MeiliSearch
4. **Download** movies and series using aria2's powerful download engine
5. **Monitor** your favorite series for automatic new episode downloads
